# react-webpack-js

## Technical

Source repository React with react-webpack.

> Node: **v16.18.0+**

> NPM: **v9.1.2**

### Use

> Clone repository

```js
git clone https://github.com/vkhangstack/react-webpack-js.git
```

> Install dependencies

```json
npm install
```

> Run app

```json
npm start
```

> Build production

```json
npm run build
```
